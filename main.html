Welcome to the Favicon Fetcher API

This API allows you to fetch the favicon of any website by simply providing its URL.

How to Use

GET /favicon

Fetch the favicon of the specified website.

Request:
GET /favicon?url=<website_url>

Parameters:
- url (required): The URL of the website you want to fetch the favicon from.

Example:
/favicon?url=www.google.com

Response

The API will return the favicon image of the specified website.
